# Mutual Fund Data Extraction & Analysis Project

## 🎯 Project Objective
This project demonstrates the ability to extract unstructured financial data from PDF factsheets and transform it into a standardized, analytical dataset. The goal is to compare performance (1Y/3Y returns) against costs (Expense Ratios) across Equity, Debt, and ETF categories.

## 🛠️ Tools & Skills
- **Data Extraction:** Manual and automated parsing of financial PDFs.
- **Data Cleaning:** Excel-based normalization of inconsistent naming conventions and formatting.
- **Analysis:** Quantitative grouping and aggregation to identify market trends.
- **Documentation:** Professional reporting for stakeholder review.

## 📊 Key Findings
- **Expense Advantage:** ETFs demonstrated a 75% lower cost structure than active equity funds.
- **Alpha Generation:** Active Equity managers in this dataset outperformed benchmarks by ~4% over a 3-year horizon.
- **Sectoral Risk:** Identified significant drawdowns in IT-specific funds despite a generally bullish market.

## 📁 Repository Contents
- `data/`: Contains the cleaned dataset with 20+ fund entries.
- `analysis/`: Summary tables and categorical breakdowns.
- `docs/`: Full project report and methodology.
